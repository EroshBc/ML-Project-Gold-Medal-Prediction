# ML-Project-Gold-Medal-Prediction

### Objective
predict how many medals a country will win based on historical and current data.

### Project Overview
- Form a hypothesis.
- Find and explore the data.
- Reshape the data to predict your target.
- Clean the data for ML.
- Pick an error metric.
- Split your data.
- Train a model.

### Dataset
- Dataset : 120 years of Olypic history
- Source : https://www.kaggle.com/datasets/heesoo37/120-years-of-olympic-history-athletes-and-results
- Records include :
    - ID
    - Name
    - Sex
    - Age
    - Height
    - Weight
    - Team
    - NOC
    - Games
    - Medals
  

Used linear regression
